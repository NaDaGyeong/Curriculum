프로그래밍적 의미에서, 혹은 수학적 의미에서 변수는 1. 값을 저장하는, 2. 이름이 주어진 공간을 의미합니다. 예컨대 100이라는 숫자를 특정한 공간에 넣고 해당 공간에 score이라는 이름을 부여해주면, score는 100이라는 값을 갖는 변수가 됩니다.

```python
score=100
```
